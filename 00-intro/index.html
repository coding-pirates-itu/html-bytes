Tags

  • <tag-name ...> - start
  • </tag-name> - end
  • <tag-name .../> - self-closing
  • Attributes: name="value"; quotes: " or '

Example: <a href="https://dr/dk">DR</a>

  1. <
  2. a
  3. href
  4. ="https://dr.dk"
  5. >
  6. DR
  7. </
  8. a>

Nested tags

<p> <a> <img/> </a> </p>

Case sensitivity

<P> vs <p>
